# Getting Started with Astro

Astro is a powerful static site generator that allows you to build fast websites with your favorite frameworks. In this post, we'll cover the basics.

## Why Astro?

- **Performance**: Only ships JavaScript for interactive components
- **Framework Agnostic**: Use React, Vue, Svelte, or plain HTML
- **Content Focused**: Great for blogs and documentation

## Installation

```bash
npm create astro@latest
```

## Basic Structure

Astro projects have a simple structure:

- `src/pages/` for routes
- `src/components/` for reusable components
- `src/content/` for content collections
